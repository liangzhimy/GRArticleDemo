WKWebView - Loading local file, assets and css.

Local files and assets load correctly if baseUrl is set.

A ScriptMessageHandler "MyObserver" is added when the WKWebView is configured at start up. The "Call Javascript" button calls the javascript function testJS() in this html file.

window.webkit.messageHandlers.MyObserver.postMessage(message) calls the Objective-C method: userContentController:(WKUserContentController *)userContentController didReceiveScriptMessage:(WKScriptMessage *)message and brings up a dialog with the passed message text.
